# 🍋 Little Lemon Newsletter Subscription App

![](littleLemonAppDemo.gif)
### Features

- Newsletter subscription form with email validation
- User-friendly interface for managing subscriptions
- Responsive design across mobile devices

### Tech Stack

- **Framework**: [React Native](https://reactnative.dev/)
- **Language**: [JavaScript](https://developer.mozilla.org/es/docs/Web/JavaScript)
- **State Management**: React Hooks ([useState](https://react.dev/reference/react/useState))
- **Styling**: StyleSheet (React Native)
- **Platform**: iOS and Android

### Installation

```bash
npm install
npm start
```

### Usage

Run the app on your preferred emulator or device using the React Native CLI.
